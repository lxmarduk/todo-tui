# TODO app

This is simple TODO application made with [rataui](https://ratatui.rs/).

# Controls

Main screen:

- `a` - add new item
- `Enter` - edit selected item
- `Space` - mark item done/undone
- `UpArrow`/`DownArrow` - select item
- `q` - quit

On Edit/Add screen:

- `Enter` - confirm
- `Escape` - return back to main screen
